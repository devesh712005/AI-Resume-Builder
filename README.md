# 🚀 AI Resume Builder (MERN + Gemini AI)

A full-stack AI-powered resume builder that helps users create, enhance, and analyze resumes using Google Gemini AI.

---

## ✨ Features

- 🧠 AI Resume Summary Generator  
  Generate ATS-friendly professional summaries

- 📄 AI Job Description Enhancer  
  Convert raw experience into impactful bullet points

- 📊 Resume Analyzer (ATS Score)  
  Upload resume → get score + improvement suggestions

- 📂 AI Resume Parser  
  Convert resume text into structured JSON

- 🔐 Authentication System  
  Secure login/signup using JWT

- ☁️ Image Upload (ImageKit)  
  Upload profile images with CDN optimization

- 🗄️ MongoDB Storage  
  Store resumes and user data efficiently

---

## 🏗️ Tech Stack

### Frontend
- React.js (Vite)
- Redux Toolkit
- Tailwind CSS
- Axios

### Backend
- Node.js
- Express.js
- MongoDB + Mongoose
- JWT Authentication

### AI Integration
- Google Gemini API (`@google/genai`)

### Cloud Services
- ImageKit
- MongoDB Atlas
